## Phase 3: User Story 1 — View Dashboard Shell (Priority: P1) 🎯 MVP

**Goal**: Render a styled dashboard page at `/dashboard` with top bar, sidebar placeholder, and main placeholder in dark mode.

**Independent Test**: Run `pnpm dev`, navigate to `http://localhost:3000/dashboard`, confirm all items in `quickstart.md` acceptance checklist pass.

### Implementation for User Story 1

- [x] T006 [US1] Create `src/components/dashboard/top-bar.tsx` — Server Component; renders a full-width `<header>` with: DevStash brand text on the left, shadcn `<Input>` with `placeholder="Search items..."` in the center, shadcn `<Button variant="outline">New Collection</Button>` and `<Button>New Item</Button>` on the right; no event handlers
- [x] T007 [US1] Create `src/app/dashboard/page.tsx` — Server Component; imports `<TopBar />`; renders layout with `<TopBar />` at top, then a flex container with `<aside><h2>Sidebar</h2></aside>` and `<main><h2>Main</h2></main>`; apply Tailwind classes for full-height layout

**Checkpoint**: Navigate to `/dashboard` — top bar, "Sidebar" h2, and "Main" h2 all visible on dark background with no console errors
